    New Features
    • Email Notifications
      • You can now subscribe to emails for your favourite sources!
      • To do so, go to a Target page (e.g. here) and scroll down to the 'Get Email Notifications about this source' section.

GEMTOM v0.1.3 - Orphans


    New Features
    • Orphaned Transients!
      • The nightly transients page now contains a list of orphaned transients found that night, and links to their relevant GEMTOM pages.
      • BlackGEM is queried for new orphaned transients every day (at 10, 12, and 14 UT).
      • These transients are continuously fed through a Random Forest classifier, based on the user-submitted notes. This generates a Real/Bogus score specifically made for orphaned sources.
      • You can label orphaned transients as interesting or not ('Yes' or 'No'), along with writing custom notes; this is only saved in GEMTOM, and doesn't communicate elsewhere.
      • GEMTOM also searches its own records for past notes, so if an orphaned transient has already been noted to be a Diffraction Spike or a TNS source, that shows up on each night's page.
      • All labelled Orphaned Transients are listed on a new Transients page. Orphans can also be reviewed and re-rated here.

    • Search by TNS, IAU Name, and Gaia DR3 Designation
      • The Transients page now lets you search by TNS. The nearest target within 100 arcseconds is returned.
        If present, you're redirected automatically to the nearest target within 2 arcseconds.
      • You can also search by IAU Name (BGEM JXXX-XXX). This is a fuzzy search, which will return all objects roughly matching your query. If multiple targets match, a list will be returned.
      • You can also search by Gaia DR3 Designation. If multiple targets match, a list will be returned, with their distances.
      • Additionally, the Coordinate search has been edited to work like these alternative search methods, which avoids some bugs in the old method.
      • Finally, you can now search for an RA and Dec, and find the last time those coordinates have been observed.

    • Miscellaneous features
      • Targets can now be added to GEMTOM via a URL (/url_to_GEMTOM/[BlackGEM ID]), useful to adding from external tools.
      • Word display limit has been removed on comments. Character count remains at 3000 (roughly 500 words, or an A4 page of text)
      • Removed spurious 99-mag results from lightcurves
      • Transients: Added non-detections to a 'Full Observation Data' table
      • Transients: The 'TNS Object within 10 arcsecond' highlight now states whether BlackGEM data has been reported to TNS (it checks to see if 'BGEM' exists in the internal name)
      • Targets/Transients: Added link to BlackPEARL
      • Targets/Transients: Added link to search Vizier
      • Targets/Transients: Added link to search Aladin, which includes DESI Legacy DR10 and DECAPS DR2
      • Targets/Transients: Reduced SIMBAD search from 2 to 1 arcminute
      • Targets/Transients: Aladin inset now includes a 'Legacy DR10 Color' option (Thanks Hugo!)
      • Targets: Removed fields related to observations, which have not been set up.
      • Targets: BlackGEM Data Points now show uncertainties.
      • History: Nightly pages now have tabs for Orphaned, Gaia, and Extragalactic transients.
      • Meta: When 'Add the GEMTOM' is clicked, a message is now delivered (either that the target has been created, or already exists), and GEMTOM redirects to the new target's page.

    Bugfixes
    • If GEMTOM cannot connect to the Transient server, it should now serve an error message instead of crashing.
    • Transients: Flipped the x-axis in the 'Location on Sky' plot, in order to match the Aladin view.
    • Transients: Removed the 'Flux' columns from the 'Full Observation Data' table (unnecessarily made the table bigger). Also removed the ability to edit the cells.
    • Transients: Fixed a bug where the 10 arcsecond circle in the 'Location on Sky' plot would span the sky if the targets lay within 5 arcseconds of RA 0.
    • Transients: Fixed a bug where, if multiple TNS sources were within 10 arcseconds, the nearest one was not highlighted at the top of the page.
    • Lightcurves: Fixed the x axis so that it plots decimals of MJDs.
    • Live Feed: Fixed a bug that occurred if there was no limiting magnitude data for a transient ID.
    • History: Blank Sky Plot no longer made when there is no connection to BlackHub; system instead reports lack of connection.
    • History: Fixed a bug where nightly pages wouldn't appear if there are no detections
    • History: Fixed another bug where nightly pages would crash if there are no detections
    • History: Corrected the all-sky plot that's displayed when there's no data (it now fits with all other plots)
    • History: Massively sped up the creation of the sky plot and field stats; when a page is loaded, these will be re-created/found each time, up until 12:00UT of the following day. After that, a final version will be created and saved, and from that point on, that version will be used. Pages in the future also no longer look for field stats, and use a pre-saved empty version of the sky plot.
    • History: Sped up the plotting of the HR diagram. The main bottleneck was in the Mag Diff slider; this has now been reduced to only a few options.
