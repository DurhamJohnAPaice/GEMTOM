Version Changelogs

GEMTOM v0.1.1 Changelog


    New Features
      Nightly HR Diagram
      • Added an HR Diagram to each night's History page, with all the sources in the previous night's gaia-crossmatched catalogue.
      • These sources have a cutoff based on their Real/Bogus score and their Parallax/Error ratio.
      • You can also adjust a slider to manually filter by magnitude difference.
      • Clicking a point in the HR Diagram redirects to that Transient's GEMTOM page.
      • You can now download potential CVs for each night too!

    Bugfixes
    • Targets: Solved an issue where new targets were not being assigned into the 'Public' group, and thus could not be viewed.
    • Targets: Solved a bug where uploading a target without a BlackGEM ID would sometimes cause an error when the target was viewed.
    • History: Added the number of flags each night to the main History page.
    • Presentation: Fixed the margins on the Home page to work on Mobile.
    • Presentation: Fixed the margins and plots on the Live Feed page to work on Mobile
    • Presentation: Fixed the formatting of the plot, table, and 'Query BlackGEM' button on the History page on Mobile
    • Presentation: Fixed the formatting of the plot and the flags on the History index page on Mobile
